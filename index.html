Расчёт НМЦК Источник коэффициентов для ФОТ
Этот коэффициент используется в разделе ФОТ 2.2.
2.1. Расходы на оплату труда водителей ФОТ
Источник: «Расчёт НМЦК → 4. Коэффициенты… → Коэфф. к зарплате водителя»
Допустимый диапазон: от 1 до 1,15
Источник: «Расчёт НМЦК → 4. Коэффициенты… → Коэфф. подгот.-заключ. времени»
Источник: «Расчёт НМЦК → 3. Индексы… → Индекс потребительских цен»
Источник: «Расчёт НМЦК → 1. Исходные данные… → Пробег по контракту»
Формула: (12 * ЗП_расч * k_отпуска * часы * k_пз * IPC) / (пробег * фонд)
2.2. Расходы на оплату труда кондукторов ФОТ
Источник: «Расчёт НМЦК → 4. Коэффициенты… → Коэфф. за функции кондуктора»
Формула: k_kond * P_OTVti
2.3. Социальные отчисления ФОТ
Источник: «Расчёт НМЦК → 4. Коэффициенты… → Ставка соц. отчислений»
Формула: P_OTVti + P_OTKti
Формула: (sr * base) / 100
